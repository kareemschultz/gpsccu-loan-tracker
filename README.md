# GPSCCU Loan Tracker Pro

Advanced financial planning and analytics tool for managing your GPSCCU auto loan with strategic 6-month payment planning.

## Features

- 6-month payment strategy optimization
- Semi-annual gratuity payment tracking
- Real-time loan analytics and projections
- Payment history with CSV export
- Investment opportunity cost analysis
- Financial health monitoring
- Dark mode support

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for automatic deployment to GitHub Pages via GitHub Actions.

1. Push to main branch
2. GitHub Actions will build and deploy automatically
3. Access at: https://yourusername.github.io/gpsccu-loan-tracker

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts for visualizations
- GitHub Pages deployment
